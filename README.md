# 📚 Profe Apoia – Landing Page

Landing page desenvolvida para o projeto **Profe Apoia**, com foco em conversão de visitantes em contatos via WhatsApp.

O objetivo principal é apresentar os serviços de forma clara, transmitir confiança e facilitar o contato rápido com o cliente.

---

## 🎯 Objetivo

Criar uma página única (landing page) que:

- Apresente os serviços acadêmicos e pedagógicos
- Destaque autoridade e credibilidade
- Exiba provas sociais (feedbacks e trabalhos)
- Converta visitantes em contatos via WhatsApp

---

## 👥 Público-Alvo

- Acadêmicos (universitários)
- Pedagogos e professores

---

## 🧩 Estrutura da Página

A landing page é organizada em seções estratégicas:

### 🔝 Hero (Topo)
- Logo do Profe Apoia
- Chamada principal
- Subtítulo explicativo
- Botão de contato via WhatsApp

### ⚠️ Problema + Solução
- Apresenta as dores do público
- Posiciona o serviço como solução

### 📂 Área de Atuação (Abas)
- **Acadêmicos**
  - TCC
  - Trabalhos acadêmicos
  - Artigos científicos
  - Normas ABNT
  - Revisões

- **Pedagogos**
  - Atividades pedagógicas
  - Planejamento de aulas
  - Relatórios
  - Materiais didáticos

### 📄 Trabalhos
- Exemplos de arquivos (.docx)
- Demonstração prática dos serviços

### 🗣️ Feedbacks
- Imagens com avaliações reais de clientes
- Prova social

### ⭐ Por que confiar
- Diferenciais do serviço

### 👩‍🏫 Autoridade
- Apresentação da profissional:
  - **Elim Nascimento**
  - Pedagoga e Linguista

### ⚙️ Como funciona
- Passo a passo simples do atendimento

### 🚀 CTA Final
- Chamada forte para ação
- Botão de WhatsApp

---

## 🎨 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- Design responsivo (mobile-first)

---

## 📱 Funcionalidades

- Botão de WhatsApp fixo na tela
- Navegação simples (scroll único)
- Abas interativas (Acadêmicos / Pedagogos)
- Download de arquivos (.docx)
- Exibição de feedbacks com imagens
- Layout responsivo
